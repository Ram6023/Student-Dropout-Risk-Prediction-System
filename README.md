## 🏛️ Project Overview
Developed under the **Centre of Excellence (COE)** program, this system is a mission-critical tool for academic institutions. The platform leverages machine learning to proactively identify students at risk of attrition, enabling institutions to implement targeted intervention strategies.

Rebranded for maximum institutional clarity, the platform features a high-fidelity **Multi-Page Architecture** and the **Autumn Academy** design system, providing a premium, focused diagnostic experience for administrative users.

---

## 🚀 Core Modules (Multi-Page Architecture)

The platform is structured into six independent, high-performance modules:

1.  **🏠 Home**: Central landing node with project overview and feature diagnostics.
2.  **🔮 Prediction**: Neural risk scanning engine for individual student trajectories.
3.  **📊 Dashboard**: Comprehensive analytics command center for cohort-level trends.
4.  **💡 Insights**: AI-driven diagnostic explanations and intervention vectors.
5.  **📥 Upload**: Batch ingestion protocol for massive institutional data audits.
6.  **📜 About**: Institutional genesis and program development background.

---

## ✨ Enterprise Features

- **🎯 Neural Risk Forecasting**: Predictive scanning with high-confidence attrition probability scores.
- **🍂 Autumn Academy Design**: Sophisticated Amber, Rose, and Obsidian palette for professional institutional utility.
- **🔍 360° Analytical Dashboard**: Real-time visualization of risk clusters and success index metrics.
- **🛡️ Custom Branding**: Integrated high-fidelity logo and institutional watermarks throughout the platform.
- **💎 Glassmorphism 2.0**: Premium UI with progressive blurs, smooth PageTransitions, and modern typography.
- **⚡ Reactive Performance**: Optimized with Vite for sub-second module switching and real-time data updates.

---

## 🛠️ Technological Infrastructure

### **Frontend & Visual Systems**
- **Core**: React 18 + Vite (Multi-page platform architecture).
- **Navigation**: React Router DOM (Independent page routing).
- **Animations**: Framer Motion (PageTransitions and staggered entrance reveals).
- **Design System**: Tailwind CSS v4 (Custom Autumn Academy tokens).
- **Icons**: Lucide React (Institutional iconography).

### **Backend & Intelligence Layer**
- **Framework**: FastAPI (High-performance asynchronous Python REST API).
- **ML Engine**: Scikit-Learn (Weighted predictive modeling).
- **Data Logic**: Pandas & NumPy for concurrent metadata processing.
- **Integrity**: Pydantic for strict schema enforcement.

## 🚀 Quick Setup

### **1. Repository Initialization**
```bash
git clone https://github.com/Ram6023/Student-Dropout-Risk-Prediction-System.git
cd Student-Dropout-Risk-Prediction-System
```

### **2. Backend Provisioning**
```bash
# Recommended: Create a virtual environment
pip install -r backend/requirements.txt
uvicorn backend.main:app --reload
```

### **3. Frontend Deployment (Dev)**
```bash
cd frontend
npm install
npm run dev
```
